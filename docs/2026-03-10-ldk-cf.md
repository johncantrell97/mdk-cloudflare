# ldk-cf Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Rust WASM library that wraps the `lightning` crate for ephemeral Lightning node operation on Cloudflare Workers.

**Architecture:** Rust crate compiled to `wasm32-unknown-unknown` via `wasm-bindgen`. All I/O delegated to JavaScript through duck-typed extern interfaces (JsStore, JsSocket, JsFetcher). LDK traits implemented with sync-to-async bridging (buffer+flush pattern). TypeScript wrapper package provides Cloudflare-native bindings.

**Tech Stack:** Rust, wasm-bindgen, LDK (`lightning` crate from moneydevkit/rust-lightning lsp-0.2.0), wasm-pack, TypeScript, Cloudflare Workers

**Spec:** `docs/superpowers/specs/2026-03-10-ldk-cf-design.md`

```rust
use lightning::util::logger::{Logger, Record};

use crate::JsLogFn;

/// Logger that delegates to a JavaScript callback.
/// Falls back to web_sys::console if no callback provided.
pub struct JsLogger {
    callback: Option<JsLogFn>,
}

impl JsLogger {
    pub fn new(callback: Option<JsLogFn>) -> Self {
        Self { callback }
    }
}

// Sound: WASM is single-threaded. JsLogFn is a JS object that won't cross threads.
unsafe impl Send for JsLogger {}
unsafe impl Sync for JsLogger {}

impl Logger for JsLogger {
    fn log(&self, record: Record) {
        let level = match record.level {
            lightning::util::logger::Level::Gossip => "GOSSIP",
            lightning::util::logger::Level::Trace => "TRACE",
            lightning::util::logger::Level::Debug => "DEBUG",
            lightning::util::logger::Level::Info => "INFO",
            lightning::util::logger::Level::Warn => "WARN",
            lightning::util::logger::Level::Error => "ERROR",
        };

        let msg = format!(
            "[{}:{}] {}",
            record.module_path,
            record.line,
            record.args
        );

        if let Some(ref cb) = self.callback {
            cb.log(level, &msg);
        }
        // If no callback, silently drop — minimizes WASM binary size
    }
}
```

### Task 4: CfSocketDescriptor

The `SocketDescriptor` trait requires `Eq + Hash + Clone` (NOT Send+Sync in this LDK version). The descriptor buffers outbound data synchronously; the async pump loop flushes it.

- [ ] **Step 1: Write CfSocketDescriptor**

Create `ldk-cf/src/transport.rs`:

```rust
use std::cell::{Cell, RefCell};
use std::hash::{Hash, Hasher};
use std::rc::Rc;

use lightning::ln::peer_handler::SocketDescriptor;

/// A SocketDescriptor that buffers outbound data for async flushing.
///
/// Does NOT hold a JsSocket — the async pump loop owns the socket and
/// drains this descriptor's buffer. This keeps the descriptor lightweight
/// and avoids Send/Sync issues with JS objects.
#[derive(Clone)]
pub struct CfSocketDescriptor {
    id: u64,
    outbound_buffer: Rc<RefCell<Vec<u8>>>,
    disconnected: Rc<Cell<bool>>,
}

impl CfSocketDescriptor {
    pub fn new(id: u64) -> Self {
        Self {
            id,
            outbound_buffer: Rc::new(RefCell::new(Vec::new())),
            disconnected: Rc::new(Cell::new(false)),
        }
    }

    /// Take all buffered outbound data. Called by the pump loop to flush to the socket.
    pub fn take_buffered(&self) -> Vec<u8> {
        let mut buf = self.outbound_buffer.borrow_mut();
        std::mem::take(&mut *buf)
    }

    /// Check if disconnect was requested.
    pub fn is_disconnected(&self) -> bool {
        self.disconnected.get()
    }
}

impl SocketDescriptor for CfSocketDescriptor {
    fn send_data(&mut self, data: &[u8], _resume_read: bool) -> usize {
        if self.disconnected.get() {
            return 0;
        }
        self.outbound_buffer.borrow_mut().extend_from_slice(data);
        data.len()
    }

    fn disconnect_socket(&mut self) {
        self.disconnected.set(true);
    }
}

// Sound: WASM is single-threaded. These will never cross thread boundaries.
// Required because LDK's PeerManager needs SocketDescriptor: Send + Sync in some configurations.
unsafe impl Send for CfSocketDescriptor {}
unsafe impl Sync for CfSocketDescriptor {}

impl PartialEq for CfSocketDescriptor {
    fn eq(&self, other: &Self) -> bool {
        self.id == other.id
    }
}

impl Eq for CfSocketDescriptor {}

impl Hash for CfSocketDescriptor {
    fn hash<H: Hasher>(&self, state: &mut H) {
        self.id.hash(state);
    }
}
```

- [ ] **Step 1: Write KvPersister**

Create `ldk-cf/src/store.rs`:

```rust
use std::cell::RefCell;

use lightning::chain::chainmonitor::MonitorUpdateId;
use lightning::chain::channelmonitor::{ChannelMonitor, ChannelMonitorUpdate};
use lightning::chain::transaction::OutPoint;
use lightning::chain::Persist;
use lightning::chain::ChannelMonitorUpdateStatus;
use lightning::sign::InMemorySigner;
use lightning::util::ser::Writeable;

use crate::JsStore;

// Sound: WASM is single-threaded. Required because KvPersister is wrapped in Arc for LDK traits.
unsafe impl Send for KvPersister {}
unsafe impl Sync for KvPersister {}

/// Buffered write entry: funding outpoint key, serialized monitor bytes, update ID.
struct PendingWrite {
    key: String,
    data: Vec<u8>,
    update_id: MonitorUpdateId,
    funding_txo: OutPoint,
}

/// Persister that buffers ChannelMonitor writes synchronously,
/// then flushes them async to Cloudflare KV in the pump loop.
pub struct KvPersister {
    pending_writes: RefCell<Vec<PendingWrite>>,
    pending_deletes: RefCell<Vec<String>>,
}

impl KvPersister {
    pub fn new() -> Self {
        Self {
            pending_writes: RefCell::new(Vec::new()),
            pending_deletes: RefCell::new(Vec::new()),
        }
    }

    /// Flush all buffered writes to KV. Called from the async pump loop.
    pub async fn flush(&self, store: &JsStore) -> Result<Vec<(OutPoint, MonitorUpdateId)>, String> {
        let writes: Vec<PendingWrite> = self.pending_writes.borrow_mut().drain(..).collect();
        let deletes: Vec<String> = self.pending_deletes.borrow_mut().drain(..).collect();

        let mut flushed = Vec::new();

        for w in writes {
            store.put(&w.key, &w.data).await
                .map_err(|e| format!("KV put failed for {}: {:?}", w.key, e))?;
            flushed.push((w.funding_txo, w.update_id));
        }

        for key in deletes {
            store.delete(&key).await
                .map_err(|e| format!("KV delete failed for {}: {:?}", key, e))?;
        }

        Ok(flushed)
    }

    /// Check if there are pending writes that need flushing.
    pub fn has_pending(&self) -> bool {
        !self.pending_writes.borrow().is_empty() || !self.pending_deletes.borrow().is_empty()
    }
}

fn monitor_key(funding_txo: &OutPoint) -> String {
    format!("monitors/{}_{}", funding_txo.txid, funding_txo.index)
}

impl<ChannelSigner: lightning::sign::ecdsa::EcdsaChannelSigner> Persist<ChannelSigner> for KvPersister {
    fn persist_new_channel(
        &self,
        funding_txo: OutPoint,
        monitor: &ChannelMonitor<ChannelSigner>,
        update_id: MonitorUpdateId,
    ) -> ChannelMonitorUpdateStatus {
        let mut bytes = Vec::new();
        monitor.write(&mut bytes).expect("serialization cannot fail");
        self.pending_writes.borrow_mut().push(PendingWrite {
            key: monitor_key(&funding_txo),
            data: bytes,
            update_id,
            funding_txo,
        });
        ChannelMonitorUpdateStatus::InProgress
    }

    fn update_persisted_channel(
        &self,
        funding_txo: OutPoint,
        _update: Option<&ChannelMonitorUpdate>,
        monitor: &ChannelMonitor<ChannelSigner>,
        update_id: MonitorUpdateId,
    ) -> ChannelMonitorUpdateStatus {
        let mut bytes = Vec::new();
        monitor.write(&mut bytes).expect("serialization cannot fail");
        self.pending_writes.borrow_mut().push(PendingWrite {
            key: monitor_key(&funding_txo),
            data: bytes,
            update_id,
            funding_txo,
        });
        ChannelMonitorUpdateStatus::InProgress
    }

    fn archive_persisted_channel(&self, funding_txo: OutPoint) {
        self.pending_deletes.borrow_mut().push(monitor_key(&funding_txo));
    }
}
```

**Important:** The `Persist` trait signature varies between LDK versions. The spec says it uses `MonitorName` but the actual MDK fork may use `OutPoint` or a different parameter order. The code above uses `OutPoint` + `MonitorUpdateId` as separate parameters (common in recent LDK). Adjust based on compilation errors — the pattern (buffer + InProgress + async flush) stays the same regardless of the exact signature.

```rust
use std::cell::RefCell;
use std::collections::HashMap;

use bitcoin::Transaction;
use lightning::chain::chaininterface::{
    BroadcasterInterface, ConfirmationTarget, FeeEstimator,
};

use crate::JsFetcher;

// Sound: WASM is single-threaded. Required because these are wrapped in Arc for LDK traits.
unsafe impl Send for EsploraFeeEstimator {}
unsafe impl Sync for EsploraFeeEstimator {}
unsafe impl Send for EsploraBroadcaster {}
unsafe impl Sync for EsploraBroadcaster {}

/// Fee estimator that fetches rates from Esplora and caches them per invocation.
pub struct EsploraFeeEstimator {
    cached_rates: RefCell<HashMap<ConfirmationTarget, u32>>,
}

impl EsploraFeeEstimator {
    pub fn new() -> Self {
        Self {
            cached_rates: RefCell::new(HashMap::new()),
        }
    }

    /// Fetch fee estimates from Esplora and populate cache.
    /// Call once at the start of each Worker invocation.
    pub async fn refresh(&self, fetcher: &JsFetcher, esplora_url: &str) -> Result<(), String> {
        let url = format!("{}/fee-estimates", esplora_url);
        let result = fetcher.get_json(&url).await
            .map_err(|e| format!("Fee estimate fetch failed: {:?}", e))?;

        // Esplora returns { "1": 87.882, "2": 87.882, ... } (sat/vbyte by confirmation target)
        let json: String = js_sys::JSON::stringify(&result)
            .map_err(|_| "JSON stringify failed".to_string())?
            .into();
        let rates: HashMap<String, f64> = serde_json::from_str(&json)
            .map_err(|e| format!("Fee parse failed: {}", e))?;

        let mut cache = self.cached_rates.borrow_mut();

        // Convert sat/vbyte to sat/kw (1 vbyte = 4 weight units, 1000 weight units per kw)
        let sat_per_vbyte_to_sat_per_kw = |spv: f64| -> u32 {
            std::cmp::max((spv * 250.0) as u32, 253) // LDK minimum is 253
        };

        // Map Esplora confirmation targets to LDK targets
        let get_rate = |block_target: &str| -> u32 {
            rates.get(block_target)
                .map(|r| sat_per_vbyte_to_sat_per_kw(*r))
                .unwrap_or(253)
        };

        cache.insert(ConfirmationTarget::UrgentOnChainSweep, get_rate("1"));
        cache.insert(ConfirmationTarget::MaximumFeeEstimate, get_rate("1"));
        cache.insert(ConfirmationTarget::NonAnchorChannelFee, get_rate("3"));
        cache.insert(ConfirmationTarget::AnchorChannelFee, get_rate("6"));
        cache.insert(ConfirmationTarget::ChannelCloseMinimum, get_rate("12"));
        cache.insert(ConfirmationTarget::OutputSpendingFee, get_rate("12"));
        cache.insert(ConfirmationTarget::MinAllowedAnchorChannelRemoteFee, get_rate("25"));
        cache.insert(ConfirmationTarget::MinAllowedNonAnchorChannelRemoteFee, get_rate("25"));

        Ok(())
    }
}

impl FeeEstimator for EsploraFeeEstimator {
    fn get_est_sat_per_1000_weight(&self, target: ConfirmationTarget) -> u32 {

    }

pub struct EsploraBroadcaster {

    pending_txs: RefCell<Vec<Vec<u8>>>,
}

impl EsploraBroadcaster {
    pub fn new() -> Self {
        Self {
            pending_txs: RefCell::new(Vec::new()),
        }
    }

    /// Flush pending broadcasts to Esplora. Called from async context.
    pub async fn flush(&self, fetcher: &JsFetcher, esplora_url: &str) -> Result<(), String> {
        let txs: Vec<Vec<u8>> = self.pending_txs.borrow_mut().drain(..).collect();
        for tx_bytes in txs {
            let hex = hex::encode(&tx_bytes);
            let url = format!("{}/tx", esplora_url);
            fetcher.post_bytes(&url, hex.as_bytes()).await
                .map_err(|e| format!("Broadcast failed: {:?}", e))?;
        }
        Ok(())
    }
}

impl BroadcasterInterface for EsploraBroadcaster {
    fn broadcast_transactions(&self, txs: &[&Transaction]) {
        let mut pending = self.pending_txs.borrow_mut();
        for tx in txs {
            let mut bytes = Vec::new();
            bitcoin::consensus::encode::Encodable::consensus_encode(*tx, &mut bytes)
                .expect("tx encoding cannot fail");
            pending.push(bytes);
        }
    }
}
```

```rust
//! LSPS4 protocol message types.
//!
//! LSPS4 is MDK's custom protocol for JIT channels with offline nodes.
//! The client sends `lsps4.register_node` to get a persistent intercept SCID,
//! which is used as a route hint in invoices. When a payment arrives, the LSP
//! stores the HTLC and sends a webhook.
//!
//! Wire format: JSON-RPC payloads sent as LSPS0 custom messages (type 37913).

use serde::{Deserialize, Serialize};

pub const LSPS_MESSAGE_TYPE: u16 = 37913;
pub const LSPS4_REGISTER_NODE_METHOD: &str = "lsps4.register_node";

/// JSON-RPC request envelope
#[derive(Serialize, Debug)]
pub struct JsonRpcRequest<P: Serialize> {
    pub jsonrpc: &'static str,
    pub method: &'static str,
    pub params: P,
    pub id: String,
}

/// JSON-RPC response envelope
#[derive(Deserialize, Debug)]
pub struct JsonRpcResponse<R> {
    pub jsonrpc: String,
    pub result: Option<R>,
    pub error: Option<JsonRpcError>,
    pub id: String,
}

#[derive(Deserialize, Debug)]
pub struct JsonRpcError {
    pub code: i64,
    pub message: String,
}

/// Empty params for register_node request
#[derive(Serialize, Debug)]
pub struct RegisterNodeParams {}

/// Response from register_node
#[derive(Deserialize, Debug, Clone)]
pub struct RegisterNodeResponse {
    pub jit_channel_scid: String,
    pub lsp_cltv_expiry_delta: u32,
}

/// Parsed and validated registration result
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Lsps4Registration {
    pub intercept_scid: u64,
    pub cltv_expiry_delta: u32,
}

impl Lsps4Registration {
    /// Parse the SCID string "BBBxTTTx000" into a u64.
    pub fn from_response(resp: RegisterNodeResponse) -> Result<Self, String> {
        let scid = parse_scid_str(&resp.jit_channel_scid)?;
        Ok(Self {
            intercept_scid: scid,
            cltv_expiry_delta: resp.lsp_cltv_expiry_delta,
        })
    }

    pub fn to_bytes(&self) -> Vec<u8> {
        serde_json::to_vec(self).expect("serialization cannot fail")
    }

    pub fn from_bytes(bytes: &[u8]) -> Result<Self, String> {
        serde_json::from_slice(bytes).map_err(|e| format!("LSPS4 deser failed: {}", e))
    }
}

/// Parse "BBBxTTTxOOO" SCID format to u64.
/// Format: block_height (3 bytes) << 40 | tx_index (3 bytes) << 16 | output_index (2 bytes)
fn parse_scid_str(s: &str) -> Result<u64, String> {
    let parts: Vec<&str> = s.split('x').collect();
    if parts.len() != 3 {
        return Err(format!("Invalid SCID format: {}", s));
    }
    let block: u64 = parts[0].parse().map_err(|_| format!("Bad block in SCID: {}", s))?;
    let tx: u64 = parts[1].parse().map_err(|_| format!("Bad tx in SCID: {}", s))?;
    let out: u64 = parts[2].parse().map_err(|_| format!("Bad output in SCID: {}", s))?;

    Ok((block << 40) | (tx << 16) | out)
}

/// Build the register_node JSON-RPC request payload.
pub fn build_register_node_request(request_id: &str) -> Vec<u8> {
    let req = JsonRpcRequest {
        jsonrpc: "2.0",
        method: LSPS4_REGISTER_NODE_METHOD,
        params: RegisterNodeParams {},
        id: request_id.to_string(),
    };
    serde_json::to_vec(&req).expect("serialization cannot fail")
}

/// Parse a register_node JSON-RPC response.
pub fn parse_register_node_response(data: &[u8]) -> Result<Lsps4Registration, String> {
    let resp: JsonRpcResponse<RegisterNodeResponse> =
        serde_json::from_slice(data).map_err(|e| format!("LSPS4 response parse failed: {}", e))?;

    if let Some(err) = resp.error {
        return Err(format!("LSPS4 error {}: {}", err.code, err.message));
    }

    let result = resp.result.ok_or("LSPS4 response missing result")?;
    Lsps4Registration::from_response(result)
}
```

```rust
/// Wire message wrapper for LSPS0 custom messages.
#[derive(Debug)]
pub struct LspsMessage {
    pub payload: Vec<u8>,
}

impl Type for LspsMessage {
    fn type_id(&self) -> u16 {
        LSPS_MESSAGE_TYPE
    }
}

impl lightning::io::Writeable for LspsMessage {
    fn write<W: lightning::io::Write>(&self, writer: &mut W) -> Result<(), lightning::io::Error> {
        writer.write_all(&self.payload)
    }
}

/// Handles LSPS4 custom messages on the peer connection.
///
/// Usage:
/// 1. Call `send_register_node()` to queue an outbound message
/// 2. `get_and_clear_pending_msg()` is called by PeerManager to get queued messages
/// 3. When a response arrives, `handle_custom_message()` parses it
/// 4. Call `take_registration()` to retrieve the result
pub struct Lsps4MessageHandler {
    pending_outbound: RefCell<Vec<(PublicKey, LspsMessage)>>,
    pending_response: RefCell<Option<Result<Lsps4Registration, String>>>,
    request_counter: RefCell<u64>,
}

impl Lsps4MessageHandler {
    pub fn new() -> Self {
        Self {
            pending_outbound: RefCell::new(Vec::new()),
            pending_response: RefCell::new(None),
            request_counter: RefCell::new(0),
        }
    }

    /// Queue a register_node request to be sent to the LSP.
    pub fn send_register_node(&self, lsp_pubkey: PublicKey) {
        let mut counter = self.request_counter.borrow_mut();
        *counter += 1;
        let request_id = format!("ldk-cf-{}", *counter);
        let payload = build_register_node_request(&request_id);

        self.pending_outbound.borrow_mut().push((
            lsp_pubkey,
            LspsMessage { payload },
        ));
    }

    /// Take the registration result (if a response has been received).
    pub fn take_registration(&self) -> Option<Result<Lsps4Registration, String>> {
        self.pending_response.borrow_mut().take()
    }
}

impl CustomMessageReader for Lsps4MessageHandler {
    type CustomMessage = LspsMessage;

    fn read<R: lightning::io::Read>(
        &self,
        message_type: u16,
        buffer: &mut R,
    ) -> Result<Option<LspsMessage>, msgs::DecodeError> {
        if message_type == LSPS_MESSAGE_TYPE {
            let mut payload = Vec::new();
            buffer.read_to_end(&mut payload)
                .map_err(|_| msgs::DecodeError::Io(lightning::io::ErrorKind::Other))?;
            Ok(Some(LspsMessage { payload }))
        } else {
            Ok(None)
        }
    }
}

impl CustomMessageHandler for Lsps4MessageHandler {
    fn handle_custom_message(
        &self,
        msg: LspsMessage,
        _sender_node_id: PublicKey,
    ) -> Result<(), msgs::LightningError> {
        // Try to parse as register_node response
        let result = parse_register_node_response(&msg.payload);
        *self.pending_response.borrow_mut() = Some(result);
        Ok(())
    }

    fn get_and_clear_pending_msg(&self) -> Vec<(PublicKey, Self::CustomMessage)> {
        self.pending_outbound.borrow_mut().drain(..).collect()
    }

    fn provided_node_features(&self) -> NodeFeatures {
        NodeFeatures::empty()
    }

    fn provided_init_features(&self, _their_node_id: PublicKey) -> InitFeatures {
        InitFeatures::empty()
    }

    fn peer_disconnected(&self, _their_node_id: PublicKey) {}

    fn peer_connected(
        &self,
        _their_node_id: PublicKey,
        _msg: &msgs::Init,
        _inbound: bool,
    ) -> Result<(), ()> {
        Ok(())
    }
}
```

```rust
use lightning::events::{Event, PaymentPurpose};
use lightning::ln::channelmanager::PaymentId;
use serde::Serialize;

use crate::types::ChannelManager;

/// A payment that was successfully claimed during this invocation.
#[derive(Debug, Serialize)]
pub struct ClaimedPayment {
    pub payment_hash: String,
    pub amount_msat: u64,
}

/// Process all pending events from the ChannelManager.
///
/// Returns a list of claimed payments. Events that don't require action
/// (ChannelReady, ChannelClosed, etc.) are logged and acknowledged.
///
/// This is called in the pump loop after each round of peer message processing.
///
/// Note: LDK's `process_pending_events` takes `FnMut(Event)` (void return) in most
/// versions. If the MDK fork uses a Result-based closure, change the signature to
/// `|event| -> Result<(), ReplayEvent>` and return `Ok(())`.
pub fn process_events(
    channel_manager: &ChannelManager,
    sweepable_outputs: &mut Vec<Vec<u8>>,
) -> Vec<ClaimedPayment> {
    let mut claimed = Vec::new();

    channel_manager.process_pending_events(&mut |event| {
        match event {
            Event::PaymentClaimable {
                payment_hash,
                purpose,
                amount_msat,
                ..
            } => {
                // Extract preimage from purpose — auto-derived by LDK for
                // payments created with create_inbound_payment()
                if let Some(preimage) = purpose.preimage() {
                    channel_manager.claim_funds(preimage);
                    claimed.push(ClaimedPayment {
                        payment_hash: hex::encode(payment_hash.0),
                        amount_msat,
                    });
                }
                // If no preimage available, the event will replay next time
            }
            Event::PaymentClaimed { .. } => {
                // Payment fully settled — already tracked in PaymentClaimable
            }
            Event::OpenChannelRequest {
                temporary_channel_id,
                counterparty_node_id,
                funding_satoshis,
                ..
            } => {
                // Accept inbound channels from LSP (JIT channel opens).
                // manually_accept_inbound_channels = true requires this.
                let _ = channel_manager.accept_inbound_channel(
                    &temporary_channel_id,
                    &counterparty_node_id,
                    0, // user_channel_id — we don't track it
                );
            }
            Event::HTLCIntercepted {
                intercept_id,
                requested_next_hop_scid,
                expected_outbound_amount_msat,
                payment_hash,
                ..
            } => {
                // Forward intercepted HTLCs — this is the JIT channel flow.
                // The LSP intercepted a payment destined for us and is forwarding it.
                let _ = channel_manager.forward_intercepted_htlc(
                    intercept_id,
                    &payment_hash,
                    requested_next_hop_scid,
                    expected_outbound_amount_msat,
                );
            }
            Event::SpendableOutputs { outputs, .. } => {
                // Persist for out-of-band sweeping
                for output in outputs {
                    let mut bytes = Vec::new();
                    lightning::util::ser::Writeable::write(&output, &mut bytes)
                        .expect("serialization cannot fail");
                    sweepable_outputs.push(bytes);
                }
            }
            Event::ChannelReady { .. } => {}
            Event::ChannelClosed { .. } => {}
            _ => {
                // Acknowledge unknown events to prevent replay
            }
        }
    });

    claimed
}
```

```rust
/// An ephemeral Lightning node that restores from persisted state,
/// processes pending work, and shuts down.
pub struct EphemeralNode {
    channel_manager: Arc<ChannelManager>,
    chain_monitor: Arc<ChainMonitor>,
    peer_manager: Arc<PeerManager>,
    keys_manager: Arc<KeysManager>,
    persister: Arc<KvPersister>,
    fee_estimator: Arc<EsploraFeeEstimator>,

    logger: Arc<JsLogger>,
    lsps4_handler: Arc<Lsps4MessageHandler>,

impl EphemeralNode {
    /// Restore node from persisted state in KV, or create a fresh node.
    pub async fn restore(
        store: &JsStore,
        fetcher: &JsFetcher,
        mnemonic: &str,
        network: Network,
        esplora_url: String,
        log_fn: Option<JsLogFn>,
    ) -> Result<Self, JsValue> {
        let logger = Arc::new(JsLogger::new(log_fn));
        let fee_estimator = Arc::new(EsploraFeeEstimator::new());
        let broadcaster = Arc::new(EsploraBroadcaster::new());
        let persister = Arc::new(KvPersister::new());
        let lsps4_handler = Arc::new(Lsps4MessageHandler::new());

        // Derive seed from mnemonic
        let mnemonic = bip39::Mnemonic::parse(mnemonic)
            .map_err(|e| JsValue::from_str(&format!("Invalid mnemonic: {}", e)))?;
        let seed_bytes = mnemonic.to_seed("");
        let mut seed = [0u8; 32];
        seed.copy_from_slice(&seed_bytes[..32]);

        let now = js_sys::Date::now() as u64;
        let keys_manager = Arc::new(KeysManager::new(&seed, now / 1000, (now % 1000) as u32));

        // Refresh fee estimates
        fee_estimator.refresh(fetcher, &esplora_url).await
            .map_err(|e| JsValue::from_str(&e))?;

        // Load channel monitors from KV
        let monitor_keys = store.list("monitors/").await
            .map_err(|e| JsValue::from_str(&format!("List monitors failed: {:?}", e)))?;
        let monitor_key_list: Vec<String> = serde_wasm_bindgen::from_value(monitor_keys)
            .unwrap_or_default();

        let mut channel_monitors = Vec::new();
        for key in &monitor_key_list {
            if let Ok(data) = store.get(key).await {
                if !data.is_null() && !data.is_undefined() {
                    let bytes: Vec<u8> = serde_wasm_bindgen::from_value(data)
                        .map_err(|e| JsValue::from_str(&format!("Monitor deser failed: {}", e)))?;
                    let mut reader = lightning::io::Cursor::new(&bytes);
                    let (_, monitor) = <(_, lightning::chain::channelmonitor::ChannelMonitor<_>)>::read(
                        &mut reader,
                        (&*keys_manager, &*keys_manager),
                    ).map_err(|e| JsValue::from_str(&format!("Monitor decode failed: {:?}", e)))?;
                    channel_monitors.push(monitor);
                }
            }
        }

        // Build ChainMonitor
        let chain_monitor: Arc<ChainMonitor> = Arc::new(LdkChainMonitor::new(
            None::<Arc<dyn lightning::chain::Filter + Send + Sync>>,
            broadcaster.clone(),
            logger.clone(),
            fee_estimator.clone(),
            persister.clone(),
        ));

        // Restore or create ChannelManager
        let channel_manager = if let Ok(cm_data) = store.get("channel_manager").await {
            if !cm_data.is_null() && !cm_data.is_undefined() {
                let bytes: Vec<u8> = serde_wasm_bindgen::from_value(cm_data)
                    .map_err(|e| JsValue::from_str(&format!("CM deser failed: {}", e)))?;

                let mut monitor_refs: Vec<&mut lightning::chain::channelmonitor::ChannelMonitor<_>> =
                    channel_monitors.iter_mut().collect();

                let read_args = ChannelManagerReadArgs::new(
                    keys_manager.clone(),
                    keys_manager.clone(),
                    keys_manager.clone(),
                    fee_estimator.clone(),
                    chain_monitor.clone(),
                    broadcaster.clone(),
                    Arc::new(IgnoringMessageHandler {}), // Router
                    Arc::new(IgnoringMessageHandler {}), // MessageRouter
                    logger.clone(),
                    create_user_config(),
                    monitor_refs,
                );

                let mut reader = lightning::io::Cursor::new(&bytes);
                let (_, cm) = <(_, ChannelManager)>::read(&mut reader, read_args)
                    .map_err(|e| JsValue::from_str(&format!("CM decode failed: {:?}", e)))?;
                Arc::new(cm)
            } else {
                // Fresh node — no persisted state
                Self::create_fresh_channel_manager(
                    &keys_manager, &fee_estimator, &chain_monitor,
                    &broadcaster, &logger, network,
                )
            }
        } else {
            Self::create_fresh_channel_manager(
                &keys_manager, &fee_estimator, &chain_monitor,
                &broadcaster, &logger, network,
            )
        };

        // Register monitors with ChainMonitor
        for monitor in channel_monitors {
            let funding_txo = monitor.get_funding_txo().0;
            chain_monitor.watch_channel(funding_txo, monitor)
                .map_err(|_| JsValue::from_str("Failed to watch channel"))?;
        }

        // Build PeerManager
        // Note: Exact PeerManager::new() signature depends on LDK version.
        // The MDK fork may differ. Adjust parameters during compilation.
        let peer_manager = Arc::new(LdkPeerManager::new(
            MessageHandler {
                chan_handler: channel_manager.clone(),
                route_handler: Arc::new(IgnoringMessageHandler {}),
                onion_message_handler: Arc::new(IgnoringMessageHandler {}),
            },
            (now / 1000) as u32,
            &seed, // ephemeral_random_data
            logger.clone(),
            lsps4_handler.clone(), // custom_message_handler
            keys_manager.clone(),
        ));

        Ok(Self {
            channel_manager,
            chain_monitor,
            peer_manager,
            keys_manager,
            persister,
            fee_estimator,
            broadcaster,
            logger,
            lsps4_handler,
            esplora_url,
            network,
        })
    }

    fn create_fresh_channel_manager(
        keys_manager: &Arc<KeysManager>,
        fee_estimator: &Arc<EsploraFeeEstimator>,
        chain_monitor: &Arc<ChainMonitor>,
        broadcaster: &Arc<EsploraBroadcaster>,
        logger: &Arc<JsLogger>,
        network: Network,
    ) -> Arc<ChannelManager> {
        let best_block = BestBlock::new(bitcoin::blockdata::constants::genesis_block(network).header.block_hash(), 0);
        Arc::new(ChannelManager::new(
            fee_estimator.clone(),
            chain_monitor.clone(),
            broadcaster.clone(),
            Arc::new(IgnoringMessageHandler {}), // Router
            Arc::new(IgnoringMessageHandler {}), // MessageRouter
            logger.clone(),
            keys_manager.clone(),
            keys_manager.clone(),
            keys_manager.clone(),
            create_user_config(),
            lightning::chain::ChainParameters {
                network,
                best_block,
            },
            (js_sys::Date::now() / 1000.0) as u32,
        ))
    }

    /// Connect to LSP, run pump loop, claim payments, persist, disconnect.
    pub async fn wake_and_claim(
        &self,
        store: &JsStore,
        fetcher: &JsFetcher,
        connect_fn: &JsConnectFn,
        lsp_pubkey: PublicKey,
        lsp_host: &str,
        lsp_port: u16,
    ) -> Result<Vec<ClaimedPayment>, JsValue> {
        // Sync chain state before connecting
        self.sync_chain(store, fetcher).await?;

        // Connect to LSP
        let socket = connect_fn.connect(lsp_host, lsp_port).await
            .map_err(|e| JsValue::from_str(&format!("TCP connect failed: {:?}", e)))?;

        let mut descriptor = CfSocketDescriptor::new(1);

        // Initiate BOLT 8 handshake
        self.peer_manager.new_outbound_connection(
            lsp_pubkey,
            descriptor.clone(),
            None,
        ).map_err(|e| JsValue::from_str(&format!("Outbound connection failed: {:?}", e)))?;

        // Run pump loop
        let claimed = self.pump_loop(&socket, &mut descriptor, store).await?;

        // Persist final state
        self.persist_all(store, fetcher).await?;

        // Disconnect
        socket.close();
        self.peer_manager.disconnect_all_peers();

        Ok(claimed)
    }

    /// The pump loop: flush outbound, read inbound, process events, repeat.
    async fn pump_loop(
        &self,
        socket: &JsSocket,
        descriptor: &mut CfSocketDescriptor,
        store: &JsStore,
    ) -> Result<Vec<ClaimedPayment>, JsValue> {
        let mut all_claimed = Vec::new();
        let mut sweepable_outputs = Vec::new();
        let start = js_sys::Date::now();
        let mut last_event_time = start;

        loop {
            let now = js_sys::Date::now();

            // Safety timeout: 30 seconds
            if now - start > 30_000.0 {
                break;
            }

            // Quiet timeout: 5 seconds since last event
            if now - last_event_time > 5_000.0 && !all_claimed.is_empty() {
                break;
            }

            // Flush outbound buffer to socket
            let outbound = descriptor.take_buffered();
            if !outbound.is_empty() {
                socket.write(&outbound).await
                    .map_err(|e| JsValue::from_str(&format!("Socket write failed: {:?}", e)))?;
            }

            // Read inbound data from socket
            let inbound = socket.read().await
                .map_err(|e| JsValue::from_str(&format!("Socket read failed: {:?}", e)))?;

            if inbound.is_null() || inbound.is_undefined() {
                break; // Socket closed
            }

            let bytes: Vec<u8> = js_sys::Uint8Array::new(&inbound).to_vec();
            if bytes.is_empty() {
                break; // Socket closed
            }

            // Feed data to PeerManager
            let _ok = self.peer_manager.read_event(descriptor, &bytes)
                .map_err(|e| JsValue::from_str(&format!("read_event failed: {:?}", e)))?;

            // Process peer messages
            self.peer_manager.process_events();

            // Process LDK events (claim payments, etc.)
            let claimed = events::process_events(
                &self.channel_manager,
                &mut sweepable_outputs,
            );

            if !claimed.is_empty() {
                last_event_time = js_sys::Date::now();
                all_claimed.extend(claimed);
            }

            // Flush any pending monitor writes
            let flushed = self.persister.flush(store).await
                .map_err(|e| JsValue::from_str(&e))?;
            for (funding_txo, update_id) in flushed {
                let _ = self.chain_monitor.channel_monitor_updated(funding_txo, update_id);
            }

            // Check for disconnect
            if descriptor.is_disconnected() {
                break;
            }
        }

        // Persist sweepable outputs if any
        if !sweepable_outputs.is_empty() {
            let bytes = serde_json::to_vec(&sweepable_outputs)
                .map_err(|e| JsValue::from_str(&format!("Sweep serialize failed: {}", e)))?;
            store.put("sweepable_outputs", &bytes).await
                .map_err(|e| JsValue::from_str(&format!("Sweep persist failed: {:?}", e)))?;
        }

        Ok(all_claimed)
    }

    /// Persist ChannelManager and flush remaining monitors.
    async fn persist_all(&self, store: &JsStore, fetcher: &JsFetcher) -> Result<(), JsValue> {
        // Flush any remaining monitor writes
        let flushed = self.persister.flush(store).await
            .map_err(|e| JsValue::from_str(&e))?;
        for (funding_txo, update_id) in flushed {
            let _ = self.chain_monitor.channel_monitor_updated(funding_txo, update_id);
        }

        // Serialize and persist ChannelManager (must happen AFTER monitor flush)
        let mut cm_bytes = Vec::new();
        lightning::util::ser::Writeable::write(&*self.channel_manager, &mut cm_bytes)
            .expect("CM serialization cannot fail");
        store.put("channel_manager", &cm_bytes).await
            .map_err(|e| JsValue::from_str(&format!("CM persist failed: {:?}", e)))?;

        // Flush any pending broadcasts
        self.broadcaster.flush(fetcher, &self.esplora_url).await
            .map_err(|e| JsValue::from_str(&e))?;

        Ok(())
    }

    /// Register with LSP and create a JIT invoice.
    pub async fn register_and_invoice(
        &self,
        store: &JsStore,
        fetcher: &JsFetcher,
        socket: &JsSocket,
        descriptor: &mut CfSocketDescriptor,
        lsp_pubkey: PublicKey,
        amount_sats: u64,
        expiry_secs: u32,
    ) -> Result<(String, Lsps4Registration), JsValue> {
        // Send register_node request
        self.lsps4_handler.send_register_node(lsp_pubkey);

        // Pump until we get a response
        let registration = self.pump_until_lsps4_response(socket, descriptor).await?;

        // Persist SCID
        store.put("lsps4_scid", &registration.to_bytes()).await
            .map_err(|e| JsValue::from_str(&format!("SCID persist failed: {:?}", e)))?;

        // Create invoice
        let currency = match self.network {
            Network::Bitcoin => lightning_invoice::Currency::Bitcoin,
            Network::Testnet => lightning_invoice::Currency::BitcoinTestnet,
            Network::Regtest => lightning_invoice::Currency::Regtest,
            _ => lightning_invoice::Currency::Signet,
        };

        let invoice = crate::lsps4::invoice::create_jit_invoice(
            &self.channel_manager,
            lsp_pubkey,
            registration.intercept_scid,
            registration.cltv_expiry_delta,
            amount_sats,
            expiry_secs,
            currency,
        ).map_err(|e| JsValue::from_str(&e))?;

        self.persist_all(store, fetcher).await?;

        Ok((invoice, registration))
    }

    async fn pump_until_lsps4_response(
        &self,
        socket: &JsSocket,
        descriptor: &mut CfSocketDescriptor,
    ) -> Result<Lsps4Registration, JsValue> {
        let start = js_sys::Date::now();

        loop {
            if js_sys::Date::now() - start > 15_000.0 {
                return Err(JsValue::from_str("LSPS4 registration timed out"));
            }

            // Flush + read + process (same as pump_loop but checking for LSPS4 response)
            let outbound = descriptor.take_buffered();
            if !outbound.is_empty() {
                socket.write(&outbound).await
                    .map_err(|e| JsValue::from_str(&format!("Write failed: {:?}", e)))?;
            }

            let inbound = socket.read().await
                .map_err(|e| JsValue::from_str(&format!("Read failed: {:?}", e)))?;

            if !inbound.is_null() && !inbound.is_undefined() {
                let bytes: Vec<u8> = js_sys::Uint8Array::new(&inbound).to_vec();
                if !bytes.is_empty() {
                    let _ = self.peer_manager.read_event(descriptor, &bytes);
                }
            }

            self.peer_manager.process_events();

            // Check for LSPS4 response
            if let Some(result) = self.lsps4_handler.take_registration() {
                return result.map_err(|e| JsValue::from_str(&e));
            }
        }
    }

    pub fn node_id(&self) -> String {
        self.channel_manager.get_our_node_id().to_string()
    }

    /// Sync chain state via Esplora Confirm interface.
    pub async fn sync_chain(
        &self,
        store: &JsStore,
        fetcher: &JsFetcher,
    ) -> Result<(), JsValue> {
        // Load last-seen block
        let last_block_data = store.get("last_block").await.ok();
        let last_height: u32 = last_block_data
            .and_then(|v| {
                if v.is_null() || v.is_undefined() { None }
                else { serde_wasm_bindgen::from_value::<serde_json::Value>(v).ok() }
            })
            .and_then(|v| v.get("height")?.as_u64())
            .unwrap_or(0) as u32;

        // Fetch current tip from Esplora
        let tip_data = fetcher.get_json(&format!("{}/blocks/tip/height", self.esplora_url)).await
            .map_err(|e| JsValue::from_str(&format!("Tip fetch failed: {:?}", e)))?;
        let tip_height: u32 = serde_wasm_bindgen::from_value(tip_data)
            .map_err(|e| JsValue::from_str(&format!("Tip parse failed: {}", e)))?;

        if tip_height <= last_height {
            return Ok(());
        }

        // For each block from last_height+1 to tip, we should ideally scan for
        // transactions spending registered outputs (channel closes, HTLC settlements).
        // For a receive-only node with a single LSP, force-closes are rare.
        // Full tx scanning via Esplora's /tx API is an optimization for later.
        // For now, update best_block — LDK will detect stale channels on its own.
        let tip_header_data = fetcher.get_json(
            &format!("{}/block-height/{}", self.esplora_url, tip_height)
        ).await.map_err(|e| JsValue::from_str(&format!("Block hash fetch failed: {:?}", e)))?;
        let tip_hash_str: String = serde_wasm_bindgen::from_value(tip_header_data)
            .map_err(|e| JsValue::from_str(&format!("Hash parse failed: {}", e)))?;

        let tip_header_hex = fetcher.get_json(
            &format!("{}/block/{}/header", self.esplora_url, tip_hash_str)
        ).await.map_err(|e| JsValue::from_str(&format!("Header fetch failed: {:?}", e)))?;
        let header_hex: String = serde_wasm_bindgen::from_value(tip_header_hex)
            .map_err(|e| JsValue::from_str(&format!("Header parse failed: {}", e)))?;
        let header_bytes = hex::decode(&header_hex)
            .map_err(|e| JsValue::from_str(&format!("Header hex failed: {}", e)))?;

        let mut cursor = lightning::io::Cursor::new(&header_bytes);
        let header: bitcoin::blockdata::block::Header = lightning::util::ser::Readable::read(&mut cursor)
            .map_err(|e| JsValue::from_str(&format!("Header decode failed: {:?}", e)))?;

        // Update best block on ChannelManager and ChainMonitor
        self.channel_manager.best_block_updated(&header, tip_height);
        self.chain_monitor.best_block_updated(&header, tip_height);

        // Persist last_block
        let last_block = serde_json::json!({
            "hash": tip_hash_str,
            "height": tip_height,
        });
        let block_bytes = serde_json::to_vec(&last_block)
            .map_err(|e| JsValue::from_str(&format!("Serialize failed: {}", e)))?;
        store.put("last_block", &block_bytes).await
            .map_err(|e| JsValue::from_str(&format!("Block persist failed: {:?}", e)))?;

        Ok(())
    }

    /// Full register + invoice flow: connect to LSP, register, create invoice, persist, disconnect.
    pub async fn connect_register_and_invoice(
        &self,
        store: &JsStore,
        fetcher: &JsFetcher,
        connect_fn: &JsConnectFn,
        lsp_pubkey: PublicKey,
        lsp_host: &str,
        lsp_port: u16,
        amount_sats: u64,
        expiry_secs: u32,
    ) -> Result<String, JsValue> {
        let socket = connect_fn.connect(lsp_host, lsp_port).await
            .map_err(|e| JsValue::from_str(&format!("TCP connect failed: {:?}", e)))?;

        let mut descriptor = CfSocketDescriptor::new(1);

        self.peer_manager.new_outbound_connection(
            lsp_pubkey, descriptor.clone(), None,
        ).map_err(|e| JsValue::from_str(&format!("Outbound conn failed: {:?}", e)))?;

        let (invoice, _registration) = self.register_and_invoice(
            store, fetcher, &socket, &mut descriptor, lsp_pubkey, amount_sats, expiry_secs,
        ).await?;

        socket.close();
        self.peer_manager.disconnect_all_peers();

        Ok(invoice)
    }

    /// Create invoice from a previously stored SCID. No LSP connection needed.
    /// Persists CM state after invoice creation (create_inbound_payment modifies CM).
    pub async fn create_invoice_from_stored_scid(
        &self,
        store: &JsStore,
        fetcher: &JsFetcher,
        lsp_pubkey: PublicKey,
        amount_sats: u64,
        expiry_secs: u32,
    ) -> Result<String, JsValue> {
        let scid_data = store.get("lsps4_scid").await
            .map_err(|e| JsValue::from_str(&format!("SCID load failed: {:?}", e)))?;
        if scid_data.is_null() || scid_data.is_undefined() {
            return Err(JsValue::from_str("No SCID registered. Call register_and_create_invoice first."));
        }
        let scid_bytes: Vec<u8> = serde_wasm_bindgen::from_value(scid_data)
            .map_err(|e| JsValue::from_str(&format!("SCID deser failed: {}", e)))?;
        let registration = Lsps4Registration::from_bytes(&scid_bytes)
            .map_err(|e| JsValue::from_str(&e))?;

        let currency = match self.network {
            Network::Bitcoin => lightning_invoice::Currency::Bitcoin,
            Network::Testnet => lightning_invoice::Currency::BitcoinTestnet,
            Network::Regtest => lightning_invoice::Currency::Regtest,
            _ => lightning_invoice::Currency::Signet,
        };

        let invoice = crate::lsps4::invoice::create_jit_invoice(
            &self.channel_manager,
            lsp_pubkey,
            registration.intercept_scid,
            registration.cltv_expiry_delta,
            amount_sats,
            expiry_secs,
            currency,
        ).map_err(|e| JsValue::from_str(&e))?;

        // Persist CM state — create_inbound_payment modifies the ChannelManager
        self.persist_all(store, fetcher).await?;

        Ok(invoice)
    }
}
```

```rust
    lsp_host: &str,
    lsp_port: u16,
    log_fn: Option<JsLogFn>,
) -> Result<JsValue, JsValue> {
    let network = parse_network(network)?;
    let lsp_pubkey = parse_pubkey(lsp_pubkey)?;

    let node = node::EphemeralNode::restore(
        store, fetcher, mnemonic, network, esplora_url.to_string(), log_fn,
    ).await?;

    let claimed = node.wake_and_claim(

        .map_err(|e| JsValue::from_str(&format!("Serialize result failed: {}", e)))
}

fn parse_network(s: &str) -> Result<Network, JsValue> {
    match s {
        "bitcoin" | "mainnet" => Ok(Network::Bitcoin),
        "testnet" => Ok(Network::Testnet),
        "signet" => Ok(Network::Signet),
        "regtest" => Ok(Network::Regtest),
        _ => Err(JsValue::from_str(&format!("Unknown network: {}", s))),
    }
}

fn parse_pubkey(s: &str) -> Result<PublicKey, JsValue> {
    s.parse::<PublicKey>()
        .map_err(|e| JsValue::from_str(&format!("Invalid pubkey: {}", e)))
}
```

```typescript
/// Wraps Cloudflare KV as JsStore for the WASM module.
export function createStore(kv: KVNamespace) {
  return {
    get: async (key: string): Promise<Uint8Array | null> => {
      const val = await kv.get(key, 'arrayBuffer')
      return val ? new Uint8Array(val) : null
    },
    put: async (key: string, value: Uint8Array): Promise<void> => {
      await kv.put(key, value)
    },
    delete: async (key: string): Promise<void> => {
      await kv.delete(key)
    },
    list: async (prefix: string): Promise<string[]> => {
      const result = await kv.list({ prefix })
      return result.keys.map((k) => k.name)
    },
  }
}

/// Wraps global fetch() as JsFetcher for the WASM module.
export function createFetcher() {
  return {
    get_json: async (url: string): Promise<unknown> => {
      const resp = await fetch(url)
      if (!resp.ok) throw new Error(`HTTP ${resp.status}: ${url}`)
      return resp.json()
    },
    post_bytes: async (url: string, body: Uint8Array): Promise<unknown> => {
      const resp = await fetch(url, { method: 'POST', body })
      if (!resp.ok) throw new Error(`HTTP ${resp.status}: ${url}`)
      return resp.json()
    },
  }
}

/// Wraps cloudflare:sockets connect() as JsConnectFn for the WASM module.
export function createConnector() {
  // Dynamic import to avoid errors in non-CF environments
  return {
    connect: async (host: string, port: number) => {
      const { connect } = await import('cloudflare:sockets' as string)
      const socket = connect({ hostname: host, port }, { secureTransport: 'off' })
      await socket.opened
      const reader = socket.readable.getReader()
      const writer = socket.writable.getWriter()
      return {
        read: async (): Promise<Uint8Array | null> => {
          const { value, done } = await reader.read()
          return done ? null : value
        },
        write: async (data: Uint8Array): Promise<void> => {
          await writer.write(data)
        },
        close: (): void => {
          socket.close()
        },
      }
    },
  }
}
```

## Implementation Notes

### Expected Compilation Challenges

The biggest challenge in this plan is **LDK's heavily parameterized generics**. The `types.rs` file and `node.rs` will require iteration to get the exact type parameters right. Here's how to approach it:

1. Start with `cargo check --target wasm32-unknown-unknown` after each file
2. When you get type errors, look at the trait bounds in the error message
3. Cross-reference with the MDK fork's actual trait definitions (not upstream docs)
4. The `IgnoringMessageHandler` fills most optional slots but may not implement all required traits in this LDK version

### Incremental Compilation Strategy

If the full crate won't compile initially (likely due to `types.rs` and `node.rs` generic issues):

1. Comment out `pub mod node;` and `pub mod types;` in `lib.rs`
2. Get everything else compiling first (logger, config, transport, store, chain, lsps4, events)
3. Then tackle `types.rs` by inspecting the actual trait bounds from compiler errors
4. Finally wire up `node.rs` with the resolved types

### What's NOT in This Plan

- **Chain sync implementation** (Esplora `Confirm` interface calls) — deferred to a follow-up task once the basic lifecycle works
- **wasm-opt post-processing** — apply after validating correctness
- **CI/CD pipeline** — add after the crate is working
- **Example Cloudflare Worker** — add after the TypeScript package is validated
- **Integration tests with real LSP** — requires signet setup, deferred
